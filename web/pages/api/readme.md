# API Endpoints

## الأرباح
- GET `/api/earnings/status` - حالة الأرباح
- POST `/api/earnings/withdraw` - طلب سحب

## المحافظ
- GET `/api/wallets/list` - قائمة المحافظ
- GET `/api/wallets/total` - الإجمالي

## المصادر
- GET `/api/sources/list` - قائمة المصادر
- GET `/api/sources/total` - الإجمالي

## النشر
- GET `/api/deploy/platforms` - المنصات
- POST `/api/deploy/all` - نشر على الكل

## النظام
- GET `/api/health` - فحص الخادم
